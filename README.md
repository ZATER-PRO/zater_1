# 关于系统

## routes文件结构
+ Login是登陆页面
+ Main是系统页面的最外层
+ Calculator文件夹是关于计算器的页面
+ HtmlEdit文件夹是关于html运行页面
+ Register文件夹是关于注册页面
+ ResetPassword文件夹是关于重置密码页面
+ PortalWeb文件夹是关于自服务样式页面
+ Crud文件夹是关于crud的页面
